# Правила проекта

## Архитектура взаимодействия

* Используется RESTful API (CRUD)

Действие | HTTP-метод | Контекст
-------- | ---------- | --------
Create   | POST       | Объект
Read     | GET        | Объект
Update   | PATCH      | Объект 
Delete   | DELETE     | Объект
  

* Входные и выходные данные (включая ошибки) являются JSON.
